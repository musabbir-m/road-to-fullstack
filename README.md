# 🚀 Journey to Fullstack  

This repository documents my learning journey through backend and frontend technologies.  
Each branch contains code, experiments, and practice projects for a specific topic.  

---

## 📚 Tech Stack Covered  

- **TypeScript** – Strongly typed JavaScript for scalable applications  
- **Node.js** – JavaScript runtime for building server-side applications  
- **Express.js** – Fast and minimalist web framework for Node.js  
- **Mongoose** – ODM for MongoDB  
- **RDBMS** – Relational database systems (SQL-based learning & practice)  
- **Redux** – State management for frontend applications  
- **Next.js** – React framework for production-grade applications  

---

## 🏗️ Repository Structure  

- `main` → Documentation and general notes  
- `mongoose` → Mongoose practice & experiments  
- `express` → Express.js practice & experiments  
- `typescript` → TypeScript learning notes & examples  
- `rdbms` → SQL and relational database practice  
- `redux` → Redux patterns and demos  
- `nextjs` → Next.js projects and experiments
